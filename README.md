### Requirements
HTML 5, CSS 3, Bootstrap, Laravel 10, PHP, MySQL

## Features
- Add records
- Edit records
- Delete records

### Installation
- Extract downloaded zip file and move to your prefered folder
- Import `sales_record.sql` into phpMyAdmin
- Run project `php artisan serve` on terminal and enjoy! :)
